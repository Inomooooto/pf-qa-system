<p>「チームルール」</p>
<h3>- 休憩時間：1010、1110、昼休憩、1400、1500</h3>
<h3>- 変数など名称は設計書のものを基準とし、別途必要な場合はリーダーに相談する</h3>
<h3>- 早退/遅刻/欠席などの報告事項はリーダーに連絡する</h3>
<h3>- コーディングは"可読性"を重視し、インデントとコメントを有効に活用する</h3>
<p>「Githubルール」</p>
<h3>- バックアップ：11時50分・15時30分、自宅作業時は作業終了時または翌日作業前</h3>
<h3>- 開発作業はhtdocs内で行い、バックアップはuserファイル内から行う</h3>
